# 大学校徽目录

请将各大学的校徽图片放在此目录下。

## 文件命名

- `njust.png` - 南京理工大学校徽
- `bit.png` - 北京理工大学校徽
- `buaa.png` - 北京航空航天大学校徽
- `seu.png` - 东南大学校徽
- `tsinghua.png` - 清华大学校徽
- `nju.png` - 南京大学校徽

## 图片要求

- 格式：`.png`（推荐，支持透明背景）或 `.jpg`
- 尺寸：建议 200x200px 到 400x400px
- 背景：透明背景或白色背景
- 质量：高清图片，确保校徽清晰可见

## 注意事项

- 如果图片不存在，会显示默认的学校图标（🏫）
- 图片会自动适配显示，保持宽高比
- 建议使用官方校徽图片，确保版权合规
