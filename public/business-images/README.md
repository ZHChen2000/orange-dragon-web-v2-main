# 业务图片目录

请将您的业务展示图片放在此目录下。

## 目录结构

### 已出版图书（image-1.png 到 image-6.png）
- 用于展示已出版的图书封面
- 建议尺寸：600px × 900px（2:3 比例）
- 格式：`.png` 或 `.jpg`

### 软件开发业务（code-1.png 到 code-3.png）
- 用于展示软件开发、系统集成等业务的代码截图
- 建议尺寸：1200px × 750px（16:10 比例）
- 格式：`.png` 或 `.jpg`

## 使用方法

### 已出版图书
将图书封面图片重命名为：
- `image-1.png`
- `image-2.png`
- `image-3.png`
- `image-4.png`
- `image-5.png`
- `image-6.png`

### 软件开发业务
将代码截图重命名为：
- `code-1.png`
- `code-2.png`
- `code-3.png`

## 支持的图片格式
- `.jpg` / `.jpeg`
- `.png`
- `.webp`

## 注意事项

- 图片会自动适配显示，建议使用高质量图片
- 如果图片不存在，会显示占位符
- 可以添加更多图片，只需在 `app/page.tsx` 中添加对应的图片卡片即可
